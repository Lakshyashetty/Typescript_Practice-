# TypeScript Setup and Execution Guide

## 1. Steps to install TypeScript
- 1) [Install Node.js from the official website:]( https://nodejs.org) 
- 2) Open Command Prompt / VS Code Terminal.
- 3) Install TypeScript globally using npm:
-  *npm i -g typescript*
- 4) Check if TypeScript is installed successfully:
- *tsc -v*
